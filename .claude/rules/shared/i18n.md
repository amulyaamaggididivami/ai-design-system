# Internationalization Rules (i18next)

## Stack

- i18next v25 + react-i18next v16
- Config and shared namespace types: `shared/src/i18n/`

## Rules

- IMPORTANT: always use `useTranslation()` hook — never call `i18next.t()` directly in components
- Translation keys: dot-separated, lowercase — `auth.login.title`, `common.buttons.submit`
- App-specific translations can extend shared namespaces in app packages
- Default language: English — all keys must have English translations as fallback
- Namespace types declared in shared for type-safe `useTranslation()`

## Dependency

`i18n/` may import from `constants/` only. No other shared folder dependencies.
